# Copywriting — Rende Mais

## Filosofia

O usuário-alvo do Rende Mais tem entre 25 e 45 anos, usa smartphone no dia a dia,
mas não tem formação financeira. Ele sabe que "render mais é melhor", mas não sabe
o que é CDI, D+0, FGC ou CDB. A skill do app é traduzir o mundo financeiro para
o português simples de quem conversa na padaria.

**Referência de tom:** Nubank — amigável, direto, usa "você", evita formal.
**Nunca:** Linguagem de banco tradicional ("conforme previsto na regulamentação...").
**Nunca:** Tom condescendente ("Como você provavelmente não sabe...").

---

## Regras Invioláveis de Copywriting

1. Use "você", nunca "o usuário" ou "o cliente"
2. Verbos no imperativo afirmativo: "Invista", "Compare", "Calcule" — nunca "Clique aqui"
3. Números sempre formatados: R$ 1.234,56 (nunca 1234.56 ou R$1234,56)
4. Taxas sempre com contexto: "13,2% ao ano" — nunca só "13,2%"
5. Primeira vez que aparecer CDI na sessão: explicar em parênteses
6. Frases curtas. Máximo 15 palavras por frase na UI.
7. Nunca use ponto de exclamação em estado de erro.
8. Emojis: proibidos na UI principal. Permitidos apenas em notificações push.

## Glossário Completo de Traduções

| Termo técnico | Como escrever no app |
|---|---|
| CDI | quanto rende por ano (CDI) |
| 112,5% do CDI | 112,5% do CDI — melhor que a média do mercado |
| D+0 | você pode sacar quando quiser |
| D+1 | disponível no próximo dia útil |
| D+30 | disponível após 30 dias |
| No vencimento | só pode sacar no final do prazo combinado |
| FGC | o governo protege seu dinheiro até R$ 250 mil |
| CDB | investimento direto em banco (tem imposto de renda) |
| LCI | investimento em crédito imobiliário — sem imposto de renda |
| LCA | investimento em crédito agrícola — sem imposto de renda |
| RDB | parecido com CDB, mesma proteção |
| Valor mínimo | mínimo para começar a investir |
| Score de recomendação | nossa pontuação de indicação |
| Liquidez | facilidade de pegar seu dinheiro de volta |
| Poupança | poupança tradicional |
| SELIC | taxa básica de juros do Brasil |
| IR regressivo | quanto menos tempo você deixar, mais imposto paga |

## Microcopy por Contexto

### Tela de Onboarding

```
Slide 0 (boas-vindas):
  Título:  "Seu dinheiro merece render mais"
  Corpo:   "A gente compara os melhores bancos do Brasil pra você.
            Sem complicação. Sem enrolação."
  Botão:   "Começar"

Slide 1 (quanto investir):
  Título:  "Quanto você quer deixar rendendo?"
  Corpo:   "Não precisa ser tudo. Começa com o que você tem disponível."
  Opção A: "Até R$ 5.000"
  Opção B: "Entre R$ 5.000 e R$ 20.000"
  Opção C: "Mais de R$ 20.000"
  Botão:   "Próximo"
  (desabilitado): "Escolhe uma opção pra continuar"  ← accessibility hint

Slide 2 (liquidez):
  Título:  "Quando você pode precisar do dinheiro?"
  Corpo:   "Não tem resposta certa. Cada situação é diferente."
  Opção A: "Quero poder sacar quando quiser"
  Opção B: "Posso esperar alguns meses"
  Opção C: "Não vou precisar por mais de 1 ano"
  Botão:   "Próximo"

Slide 3 (prioridade):
  Título:  "O que é mais importante pra você?"
  Opção A: "Quero a maior taxa possível"     ← ícone foguete
  Opção B: "Prefiro segurança e proteção"   ← ícone escudo
  Botão:   "Ver minha indicação"
```

### Tela Home (Ranking)

```
Header saudação:
  Manhã (06-12h):  "Bom dia!"
  Tarde (12-18h):  "Boa tarde!"
  Noite (18-23h):  "Boa noite!"
  Madrugada:       "Oi!"
  Com nome:        "Boa tarde, Ana!"

Subtítulo header:  "Veja onde seu dinheiro rende mais hoje"

Timestamp:         "Atualizado às 14h32"  (sem "hs", sem segundos)
Timestamp antigo:  "Dados de ontem — puxe pra atualizar"

Pull-to-refresh:   "Buscando as melhores taxas..."

Badge top card:    "★ Indicação do Rende Mais"  (apenas se score > 80)
```

### Card de Oferta

```
Taxa:              "112,5% ao ano"   (nunca "112.5% CDI" sem contexto)
Projeção mensal:   "≈ R$ 18,50/mês"  (com o símbolo ≈ para indicar estimativa)
Nota projeção:     "com R$ 1.000 investido"  (footer do card, 12px, neutral-400)
Botão CTA:         "Investir na XP"  ou  "Abrir conta na Rico"
```

### Tela de Detalhe

```
Cabeçalho taxa:    "[BANCO] paga X% ao ano"
Ganho vs poupança: "Comparado à poupança, você ganha R$ X a mais por mês"
Histórico gráfico: "Como a taxa variou nos últimos 30 dias"
Seção detalhes:    "O que você precisa saber"
Imposto:           "Paga imposto de renda" ou "Livre de imposto de renda"
FGC:               "Protegido pelo governo até R$ 250 mil"

Rodapé CTA:        "Você vai ser redirecionado para o site do banco."
```

### Calculadora

```
Campo valor:       "Quanto você quer investir?"  placeholder: "R$ 0,00"
Campo prazo:       "Por quanto tempo?"
Campo banco:       "Em qual banco?"
Resultado:         "Você terá R$ [X] no final"
Rendimento:        "Lucro de R$ [X] (Y%)"
Comparação:        "Seria R$ X a menos na poupança"
Nota IR:           "* Descontado o imposto de renda estimado"
```

### Estados de Loading / Erro / Vazio

```
Loading inicial:   "Buscando as melhores taxas..."  (abaixo do skeleton)
Sem conexão:       "Sem conexão agora. Mostrando os dados que temos."
Erro de servidor:  "Algo deu errado do nosso lado. Tenta de novo?"
   → Botão:        "Tentar de novo"
Sem resultados:    "Não encontramos opções com esse filtro."
   → Subtítulo:    "Tenta mudar os filtros lá em cima."
   → Botão:        "Ver todos"
Lista vazia:       "Ainda não temos bancos cadastrados. Em breve!"

Formulário — erros:
  CPF inválido:    "Hmm, esse CPF não tá batendo. Confere aí?"
  Email inválido:  "Isso não parece um e-mail válido."
  Senha fraca:     "Usa pelo menos 8 caracteres, tá? É pra sua segurança."
  Campo vazio:     "Preenche esse campo antes de continuar."
```

### Fluxo de Afiliado

```
Antes de redirecionar (bottom sheet ou toast):
  Título:  "Abrindo o site da [Banco]..."
  Corpo:   "Você vai ser redirecionado para criar sua conta. Pode levar alguns segundos."
  Botão:   "Continuar"  (abre URL)
  Link:    "Cancelar"

Após retorno ao app (deep link ou timer):
  Toast:   "Bem-vindo de volta! Se precisar de ajuda, a gente tá aqui."
```

### Notificações Push

```
Alerta de taxa:
  Título:  "📈 [BANCO] subiu a taxa!"
  Corpo:   "[BANCO] agora paga X% ao ano. Confere no app."

Taxa favorita mudou:
  Título:  "⚠️ A taxa da [BANCO] mudou"
  Corpo:   "Era X%, agora é Y%. Ainda é uma boa opção?"

Lembrete suave (D+7 sem abrir):
  Título:  "Seu dinheiro ainda pode render mais 💚"
  Corpo:   "Vê as taxas de hoje — tem novidade por aí."
```

## O Que Nunca Escrever

```
❌  "Conforme as condições do produto financeiro..."
❌  "O usuário deve preencher os campos obrigatórios."
❌  "Erro 404 — recurso não encontrado."
❌  "Taxa: 112.5%CDI D+0 FGC 250k"
❌  "Clique aqui"  (use verbo descritivo)
❌  "Saiba mais"   (use "Ver detalhes" ou "Entender melhor")
❌  "Obrigado!"    (após ação simples — desnecessário)
❌  "Sucesso!"     (desnecessário se a UI já mostra o resultado)
```
